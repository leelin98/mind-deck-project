- chip-round.png       (一般圓形晶圓 — Standard round wafer sketch)
- network-5g.png       (5G 專網模組 — 5G antenna/tower sketch)
- chip-edge.png        (邊緣感知晶片 — Edge AI chip sketch)
- satellite-leo.png    (低軌道衛星節點 — LEO satellite sketch)
- vehicle-sensor.png   (自動駕駛感測器 — Autonomous vehicle sensor sketch)
- chip-photonic.png    (矽光子互連 — Silicon photonics sketch)
- chip-square.png      (次世代方形晶圓 — Isometric square wafer sketch)

Recommended: transparent PNG, 200×200px minimum, sketch/line-art style
